# Challenge technique pour le poste de Développeur Full-stack JavaScript [Tawa x Rabie]


## Steps for Project Execution :

##### Client Install
```
cd frontend
npm install 
npm start 
```

##### Server Install
```
cd Back-end
npm install 
npm run dev 
```
